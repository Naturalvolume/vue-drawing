组件间传值

vue的组件间传值

组件间的传值有以下几种情况：

  • 父子组件间
  • 兄弟组件间

    父子组件间的传值方式

    父组件通过prop给子组件下发数据，子组件通过事件给父组件发送信息。
  • 父组件中定义数据
    export default {
data() {
 return {
   width: 5px;
 }
}
}

  • 父组件传递给子组件 width是在父组件中的名称，wid是传给子组件的名称

    <Tab :width="wid"></Tab>

  • 子组件用props接收数据

    export default {
props: ['wid']
}
  • 子组件向父组件发送改变宽度为20px事件
    <div @click="change"></div>

    methods() {
change() {
 this.$emit('changeWid', 20px)
}
}
  • 父组件监听子组件事件
    <Tab :width="wid" @changeWid='changeWidth'></Tab>

    export default {
export default {
data() {
 return {
   width: 5px;
 }
},
methods: {
 changeWidth(val) {
   this.width = val
 }
}
}
}
